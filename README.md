## spring-docker-demo

Spring Boot Example using Docker

## How to use

1. Navigate to `src/main/docker`.

```shell
cd src/main/docker/
```

2. Run the docker compose.

```shell
docker compose up
```

## Notes
1. This repository uses Java 17.
2. Make sure adjust the Java version inside the `Dockerfile` if another version is used.
